# Changelog

## 0.2.0-alpha — 2026-07-12

First public alpha of STEP — Skill Training & Evolution Panel.

### Highlights

- Tracks learned weapon, Defense, Unarmed, primary profession, and secondary profession skills.
- Provides compact and expanded progress views with per-skill visibility controls.
- Records skill gains with active and online time, including combat-aware weapon timing and profession activity tracking.
- Includes current-session and complete history views, chat sharing, and history clearing with confirmation.
- Adds configurable skill notifications with position, scale, sound channel, live preview, and a scrollable sound catalog.
- Supports native sounds plus optional WeakAuras and Decursive media when those addons are installed.
- Provides synchronized native AddOns options and a movable standalone configuration window.
- Includes presets and category-wide controls for visibility, logging, and notifications.
- Supports English and Brazilian Portuguese.

### Compatibility

- World of Warcraft Anniversary / Burning Crusade Classic 2.5.6.
- Interface `20506`.
